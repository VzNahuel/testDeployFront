Hay una serie de pasos a seguir:

  1. Se creo el componente "about" para hacer la vista del "acerca de"
  2. Se realiza el import de "aboutComp" en el "routingModule.ts

    3. Puedo buscar su "nombre de import" y ubicacion desde "appModule.ts

  3. Agrego el 'path' y el componente correspondiente
    4. Con esto, el routing esta completo
  4. Luego creo un enlace con la propiedad routerLink
  5. El valor del router debe ser el mismo que se uso en 'path'
  6. Finalmente, la etiqueta "router outlet" tomara la vista de mi componente
